Contribute

Needed:

Minetest is in need for a new Mac OS X maintainer. The latest version is in here; it is very outdated: https://github.com/toabi/minetest-mac/, forum topic

Donate:

You can donate money for covering hosting costs and other things.

Users:

  • The developer wiki contains useful information for advanced users.

  • The developer wiki

  • Forum / Unofficial Engine Development
  • #minetest-dev on Freenode (strictly serious work; log)
  • #minetest-delta on Freenode (more casual, not very active; log)
